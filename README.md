# E-Commerce Sales Analytics Dashboard

## Project Overview

This project analyzes e-commerce sales data using SQL Server and Power BI.

The objective was to identify key sales trends, profitability drivers, regional performance, and loss-making product categories.

---

## Tools Used

- SQL Server
- SQL Server Management Studio (SSMS)
- Power BI Desktop
- GitHub

---

## Dataset

Sample Superstore Dataset

Records: 9,994

---

## Key KPIs

- Total Sales: $2.30M
- Total Profit: $286.82K
- Total Transactions: 9,994

---

## Analysis Performed

### Category Analysis

- Technology generated the highest sales.
- Furniture generated high sales but low profit.

### Regional Analysis

- West region generated the highest sales and profit.
- South produced higher profit than Central despite lower sales.

### Product Analysis

- Copiers generated the highest profit.
- Tables generated over $206K in sales but incurred a loss of approximately $17K.

### Discount Analysis

- High discounts contributed to losses in some sub-categories.
- Discount alone was not sufficient to explain profitability differences.

---

## Dashboard Features

- KPI Cards
- Sales by Category
- Profit by Category
- Sales by Region
- Profit by Region
- Profit by Sub-Category
- Region Filter (Slicer)

---

## Business Recommendations

1. Review pricing strategy for Tables.
2. Investigate losses in Bookcases and Supplies.
3. Increase focus on high-profit categories such as Technology and Copiers.
4. Analyze why Central region has lower profit margins.
